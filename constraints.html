SQL Constraints

Constraints are rules applied to columns in a table to enforce data integrity and ensure accuracy, consistency, and reliability of the data.

The Constraint can be used in two ways

  • On Creation of The table
  • Using the alter Command

Commands

NOT NULL

Ensures a column cannot have NULL values.

Commands
Example
EmpID EmpName EmpAge
1 John Doe 30
2 Jane Smith 25
3 Michael Johnson 35
4 Emily Brown 28

UNIQUE

Ensures all values in a column are different.

Commands
Example
StudentID StudentEmail

3 michael.johnson@example.com
4 emily.brown@example.com

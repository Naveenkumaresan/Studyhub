Data Manipulation Language (DML)

DDL commands in SQL are used to manipulate data within the database. They primarily involve querying, inserting, updating, and deleting data from tables. DML commands are essential for manipulating data within a database, allowing you to retrieve, insert, update, and delete records as needed. They form the core functionality for interacting with the data stored in SQL databases.

Commands

INSERT

Adds new records into a table.

Commands
Explanation:

The INSERT command is used to add new records (rows) into a table. You specify the table name and the values to be inserted into each column. If you don't provide values for all columns, you must specify the columns explicitly. Each INSERT operation creates a new row in the table with the specified values.

UPDATE

Modifies existing records in a table.

Commands
Explanation:

The UPDATE command is used to modify existing records (rows) in a table. You specify which table to update and set new values for one or more columns. The WHERE clause is used to specify the rows to be updated based on certain conditions. If you omit the WHERE clause, all rows in the table will be updated.

Table altered.
1 row Updated.
1 row Updated.

DELETE

Removes records from a table.

Commands
Explanation:

The DELETE command is used to remove records (rows) from a table. You specify the table name and use the WHERE clause to specify the rows to be deleted based on certain conditions. If you omit the WHERE clause, all rows in the table will be deleted, resulting in an empty table.

Example:
1 row deleted.
